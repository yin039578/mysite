總執行時間:292*(s+t)
3. 呼叫API POST {{baseUrl}}/api/ap/log/clean_taiwin_data
{
    "batch_size":1000, // 每批1000
    "batch_limit":1000,// 做一千次，實際會做292次
    "batch_interval_ms":{s},// 填入算出的休息時間
    "batch_timeout_sec":10,
    "max_errors":5
}
